## Interoperability Statement

This software implements:

- **Open protocols** (Stratum V1, Stratum V2, Bitcoin RPC, CGMiner API)
- **Public APIs** documented by device manufacturers
- **Industry standards** for mining pool operation

Compatibility with third-party products is achieved through:
- Implementation of open, documented protocols
- Use of publicly available API specifications
- Reading standard system information (DMI/SMBIOS, CPUID, sysfs) for platform detection
- Reverse engineering is NOT used

Device identification is performed by matching the **user-agent string** that mining devices voluntarily send when connecting via the Stratum protocol. This is standard practice for all mining pool software.

Platform detection is performed by reading standard **DMI/SMBIOS fields** (`/sys/class/dmi/id/`), **CPUID** hypervisor leaves, and **sysfs** paths that the operating system exposes. This uses only publicly documented system interfaces. No proprietary APIs or reverse engineering is involved.
